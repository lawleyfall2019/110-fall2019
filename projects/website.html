Resource Web Site

In this assignment, you will create a website in which you provide information from or related to the topic of your paper. It should not simply be a reworking of your paper, but rather a web-friendly presentation of information related to, or drawn from, your paper. It should be published to your GitHub pages account no later than 8am on Monday, November 4.

Site Requirements

Your site must include a minimum of four pages; an introductory/landing page, two content pages, and a sources/credits page. All of the pages should use a consistent visual design (color, type, layout), and they should have navigation links to the other pages. All CSS formatting should be contained in a single linked CSS stylesheet.

Specific Requirements

  • The files for your site should be published to your GitHub Pages class project directory -- *yourusername*.github.io/igme110. (We will cover how to do this in week 9.)
  • The main page should be called index.html, must include a descriptive title, at least one relevant image, a brief overview of your site topic, navigation links to the other pages on your site, and your name.
  • The content pages should include a relevant title, written content, and at least one image.
  • The sources page should include any references from your paper that you used in creating the content for your site, as well as information about and links to the source of any images.
  • All pages must properly validate for both HTML and CSS. We will use the W3C HTML and CSS validators to check this.

You are welcome to use templates or code libraries from other sites as part of your HTML and/or CSS, but you MUST describe and provide a link to that source on your sources page. You also must make and document meaningful changes to the template so that I know you understand how it is constructed and that you've engaged in a design process.

Website Grading Rubric

Your site must be visible at the specified URL no later than 8am on Monday, November 4th.

The grading rubric for the site can be found on myCourses.
